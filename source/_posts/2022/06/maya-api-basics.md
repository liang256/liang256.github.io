### Concept
- MObjects
    - class to handle all kinds of data in Maya
    - a handle object pointing to actual data
    - typeless
- Function Sets
    - work with mobject
- Wrappers
    - allocated and deallocated by you
- Proxies
    - used to create custom maya objects (i.e. `MPxNode`, `MPxDeformerNode`)
