## Get Started
```
git clone https://github.com/liang256/liang256.github.io.git
npm install -g hexo-cli
npm install hexo-deployer-git --save
```

## Commands
### preview locally
```
hexo generate
hexo server
```
then go to http://localhost:4000/
### deploy
```
hexo clean && hexo deploy
```
